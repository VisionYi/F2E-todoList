## 專案使用的技術
- Vue、Vuex
- SCSS
- LocalStorage
- 完全手刻切版

## 開發專案使用指令
```shell
# 初始化專案
npm install

# 開啟 vue server
npm run server

# 建置發布的資料 dist
npm run build

# 自動佈署到 Github Page 上
npm run deploy
```
